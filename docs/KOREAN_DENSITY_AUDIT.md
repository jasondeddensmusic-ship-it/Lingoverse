### The Fix: Bubble Theme Everywhere
Every teach card should look like the compound word card (Screenshot 4):

3. ONE example sentence — with translation
4. If there's more to say → put it in a SEPARATE card (tip with bubble styling, or another teach card)
5. Notes: MAX 3 lines. Everything else → deepDive or split into new cards.

## ISSUE TYPE BREAKDOWN

### NOTE_WALL (328 occurrences) — THE BIGGEST PROBLEM
Teach card notes with 5-17 rendered lines. D58 says max 3 lines in note.
**Fix:** Cut note after 2-3 lines. Move the rest into deepDive, OR split into separate teach/tip cards.
**Bubble rule:** If a note has more than 3 Korean examples, each example should be its own card with the bubble layout.

### TIP_WALL (79 occurrences)
Tip cards with 11-24 rendered lines. 
**Fix:** Split into multiple tip cards. Each tip = ONE concept. Use bubble-styled Korean+English pairs.

### TIP_KOREAN_HEAVY (56 occurrences)
Tips where Korean lines outnumber English lines by 1.5x+. 
**Fix:** Every Korean line MUST have an English translation. No exceptions.

### TEACH_STREAK (57 occurrences)
4-9 teach cards in a row without any quiz/tip break.
**Fix:** After every 2-3 teach cards, insert an MC or match exercise. Max streak = 3.

### NL_SLASH (36 occurrences)
Teach nl fields cramming multiple words with slashes: "눈 / 코 / 입 / 귀"
**Fix:** Split into separate teach cards, one word per card, each with the bubble layout.

### EXAMPLE_FORMULA (7 occurrences)
Example field has "가다 → 가요" instead of a real sentence.
**Fix:** Replace with one natural Korean sentence + English translation.
