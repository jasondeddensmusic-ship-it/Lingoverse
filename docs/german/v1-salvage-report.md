## Format Migration Notes

ALL V1 content requires these conversions before integration:

1. **Field rename**: `nl` → `trg`, `en` → `src`, `exampleEn` → `exampleSrc`
2. **Add POS tags**: verb, noun, adj, adv, prep, conj, pron, num, intj, part, aux
3. **Add gender tags**: m, f, n, pl, null (for nouns/articles in gendered languages)
4. **Add funFact**: Etymology, cultural note, or memory hook (REQUIRED on all new cards)
5. **Add srcLang**: `"en"` on unit level
6. **Umlaut fix**: V1 uses ASCII (ae/oe/ue) in some cards — replace with ä/ö/ü
7. **Remove**: `phonetic` field (PP38: hidden for Latin scripts), `cognate` block (not in new format)
8. **Check PP8**: All salvaged quiz steps need hint-leak scan before integration
