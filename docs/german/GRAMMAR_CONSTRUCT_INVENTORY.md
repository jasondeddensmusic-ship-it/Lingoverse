## Vocabulary Domains by Level

| Level | New Words | Key Domains |
|-------|-----------|-------------|
| A1 | ~650 | Person, family, food/drink, home/rooms, transport/directions, shopping/clothing, time/dates, colors, numbers, countries, professions, body parts |
| A2 | ~650 | Health/illness, emotions, weather, hobbies/sports, media/tech, education, Wechselprapositionen contexts, neighborhood, relationships |
| B1 | ~1,100 | Work/career (Bewerbung, Lebenslauf), politics/society, media/news, environment/recycling, culture/history, economy, legal/Amt, feelings/opinions, travel (advanced) |
| B2 | ~1,600 | Academic/research, politics (detailed), economy/finance, ethics/society, media critique, professional communication, abstract concepts, proverbs/idioms, Amtsdeutsch, word formation |

## Communicative Functions by Level

**A1 (15)**: Introduce self/others, ask/answer personal questions, greet/farewell, tell time, state prices, order food, make purchases, ask/give directions, describe people/places, express likes/dislikes, describe routines, fill forms, write simple messages, make appointments, express ability.

**A2 (14)**: Describe past events, express plans, give/follow instructions, make comparisons, describe habits, express agreement/disagreement, describe health issues, handle travel, write informal letters, talk about weather, express wishes, ask permission, describe home, talk about education.

**B1 (15)**: Express/justify opinions, narrate stories, describe dreams/ambitions, handle travel situations, write formal letters, summarize information, express conditions/hypotheticals, give advice, negotiate plans, describe advantages/disadvantages, explain causes/consequences, report speech, give presentations, express emotions, discuss social issues.

**B2 (16)**: Present detailed arguments, negotiate/persuade, speculate about causes, discuss abstract topics, write essays/reports, adapt register, handle professional interactions, discuss current events, express irony/humor, analyze/critique texts, resolve disputes, describe complex processes, express certainty/doubt, participate in debates, write formal correspondence, give extended presentations.
