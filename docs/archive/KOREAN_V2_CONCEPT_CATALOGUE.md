## Communicative Functions (CEFR Framework)

### A1: 20 Functions
1. Introduce self (name, nationality, job)
2. Greet and farewell (formal/informal 안녕하세요/안녕)
3. Ask and answer basic questions (이름, 나이, 직업)
4. Describe people (appearance, personality)
5. Describe places and things (위치, 크기, 색깔)
6. Express likes/dislikes (좋아하다/싫어하다)
7. Ask for and give directions (어디, 왼쪽/오른쪽)
8. Order food and drink (주세요, 메뉴)
9. Handle money (가격, 얼마)
10. Tell time and dates (시간, 요일, 달)
11. Count and use numbers (하나/일, counters)
12. Describe daily routine (매일, 아침/저녁)
13. Make simple requests (주세요, -(으)세요)
14. Express existence/location (있다/없다, 에)
15. Talk about family (가족, 형제)
16. Understand simple signs and labels
17. Fill in basic forms (이름, 주소, 전화번호)
18. Express basic ability (할 수 있다)
19. React to news (정말요? 대단하다!)
20. Use polite speech level (해요체)

### A2: 20 Functions

## Unit Count Analysis (PP56 Concept-Driven)

### A1: Concepts = 270 words + 20 grammar + 20 functions
Korean A1 must cover:
- Hangul reinforcement (Foundations teaches assembly, A1 reinforces reading fluency)
- Particle system introduction (은/는, 이/가, 을/를, 에, 에서, 도)
- Three speech levels awareness (해요체 as default)
- Numbers: both 하나/둘 AND 일/이 systems + counters
- 이다/있다/없다 core patterns
- Basic survival: greetings, introductions, food, directions, time, family

Existing V1 coverage: U1-U6 (excellent quality, 93% with dialogues)
New content needed: enrichment only (pos, funFact) + quiz interleaving

**A1 units: 6** (same as V1, enriched in place)

### A2: Concepts = 636 words + 15 grammar + 20 functions
Korean A2 must cover:
- Past tense (-았/었) + future (-(으)ㄹ 거예요)
- Conditional (-(으)면)
- Comparatives (보다, 더, 가장)
- Irregular verb families (ㄷ, ㅂ)
- Social expansion: health, weather, travel, housing, school
- Casual speech (반말) introduction

Existing V1 coverage: U7-U10 (4 units, good quality)
New content needed: ~386 new teach cards for missing TOPIK A words
Estimated: 4 existing units enriched + 2-3 new units for gap words

**A2 units: 7** (4 enriched + 3 new for vocab gaps)

## TOTAL: 40 units (6 + 7 + 12 + 15)

This is concept-driven, not template-based:
- A1 (6): existing V1 is solid, just enrich
- A2 (7): V1 base + TOPIK A gap coverage
- B1 (12): V1 base + TOPIK B1 gap coverage + grammar deepening
- B2 (15): V1 base + massive TOPIK B2 vocab expansion + new grammar

Korean gets MORE units than German (36) because:
1. FSI Category IV (4x harder than Cat I)
2. B2 CEFR vocabulary is nearly 2x German B2 (3,527 vs. 1,821)
3. Script complexity requires more A1 reinforcement
4. Particle system and honorifics need dedicated practice across all levels
