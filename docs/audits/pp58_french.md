## Framework Notes

- French has 90 units: U1-30 are core communicative units (8 A1 + 8 A2 + 8 B1 + 6 B2). U31-90 are DELF vocabulary expansion batches. PP58 maps primarily to U1-30.
- French A1 spans 3 sub-levels (A1.1, A1.2, A1.3 = U1-8), providing finer granularity than German (6 units). This is favorable for function coverage.
- French has notably strong A2 coverage (100%) due to explicit tense progression (passé composé → imparfait → future → conditional).
- Health at A2 (F6) is covered only incidentally — no dedicated "chez le médecin" unit at A2, which is standard for DELF A2.
- The appointment function gap (A1) is consistent with the German mapping approach (German covers it at U6); French may address it through the calendar/time unit (U5) but without an explicit rendez-vous lesson.
