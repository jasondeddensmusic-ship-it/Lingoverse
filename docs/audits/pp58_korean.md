## Framework Notes

- Korean has 69 units vs German's 36. Units 31-69 are predominantly **vocabulary expansion batches** (TOPIK word lists), not communicative-function units. These contribute zero new function coverage.
- The **core communicative units** are U1-U30. Beyond U30, PP58 mapping is N/A (batch vocab, no functional lessons).
- Korean's unique TOPIK structure does not include some European CEFR functions (F15 "write simple messages/fill in forms", F19 "spell and give personal details") because Hangul phonics is handled via the Foundations system, not lesson units.
- Comparatives appear at B1.1 (U12) rather than A2, which is slightly late per CEFR expectations.
